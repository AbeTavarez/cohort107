JavaScript Assignment Operators

    Assignment operators assign values to JavaScript variables.

  • Assignment ( = ) assign a value to a JavaScript variable.
  • Addition Assignment ( += ) Add two numbers together and assign a value to a variable.
  • Subtraction Assignment ( -= ) performs a subtraction from two values and assign a value to a variable.
  • Multiplication Assignment ( *= ) performs a multiplication from two values and assign a value to a variable.
  • Division Assignment ( /= ) performs a division from two values and assign a value to a variable.
  • Modulus Assignment ( %= ) returns the remainder of a division and assign a value to a variable.
  • Exponentiation Assign. ( **= ) raises a number to the power of the other and assign a value to a variable.
